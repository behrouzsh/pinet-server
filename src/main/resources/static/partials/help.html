What is PiNET about?

PiNET:

piNET is a versatile tool that can be used to annotate, map and analyze a set of peptide moieties, including those with post-translationally modified sites (PTMs), identified (and quantified) by targeted (SRM type), SWATH (discovery) or related mass spectrometry based techniques. Starting from a set of peptides, and optional modifications within those peptides, piNET provides mapping into canonical UniProt protein entries, standardized annotation of modifications using PsiMOD and UNIMOD ontologies, and mapping of the sites of PTMs in the full UniProt entries. Validated modifying enzymes, including protein kinases, acetyl or methyl transferases, that are known to target these PTM sites, are mapped using iPTMnet and Signor. For phosphorylation, known kinase - target peptides are also mapped using PhosphoSitePlus, whereas DeepPhos is used to map predicted kinase â€“ target site pairs. These annotations can be visualized using versatile d3 library to provide site specific network and pathways views. The set of proteins (with PTMs mapped) can be further projected onto biological pathways using enrichment analysis and a comprehensive library of biological gene/protein sets through Enrichr, while protein abundance vectors can be compared with LINCS chemical and genetic perturbation signatures through iLINCS.

APIs

Example of Embedding pinet APIs in python:

  Python Code Snippet for Search for a List of Peptides: 


import requests
PINET_URL_BASE = 'http://pinet-server.org/pinet/api/'
PINET_URL_LINEAGE = '9606'
PEPTIDE_API_EXTENSION_1 = 'peptide/organism/'
PEPTIDE_API_EXTENSION_2 = '/peptides/'
TIMEOUT = 120
RETRIES = 3


def map_peptides(params):
    url = PINET_URL_BASE + PEPTIDE_API_EXTENSION_1 + PINET_URL_LINEAGE + PEPTIDE_API_EXTENSION_2 + params

    peptide_json = {}
    try:
        peptide_json = _request(url)
    except:
        print "error in map_peptides"
        return peptide_json

    return peptide_json

def _request(url, params=None):
    response = requests.get(url)
    return response.json()

if __name__ == '__main__':
    # Example
    # http://pinet-server.org/pinet/api/peptide/organism/9606/peptides/KSAPATGGVKKPHR,IYQYIQSR
    print map_peptides("KSAPATGGVKKPHR,IYQYIQSR")
There are programmatic ways of querying data set and attain the results in a json format:

  For Search for a List of Peptides:

http://pinet-server.org/pinet/api/peptide/organism/{organism}/peptides/{peptides} is sent for the query in the the canonivcal protein forms, e.g.
http://pinet-server.org/pinet/api/peptide/organism/9606/peptides/KSAPATGGVKKPHR,IYQYIQSR
http://pinet-server.org/pinet/api/peptide/organism/{organism}_iso/peptides/{peptides} is sent for the query in the canonivcal+iso protein form, e.g.
http://pinet-server.org/pinet/api/peptide/organism/9606_iso/peptides/KSAPATGGVKKPHR,IYQYIQSR

  For PTMProtein Query:

http://pinet-server.org/pinet/api/phosphoandptm/organism/{organism}/ptmprotein/{PTMProteinList} is sent for the query in the URL, e.g. , please note that + is encoded to %2B in the URL. This returns the phosphorylation kinase-substrate analysis, results of iPTMnet, kinase-substrate prediction and SIGNOR analysis.

Example with shorthand notation

http://pinet-server.org/pinet/api/phosphoandptm/organism/9606/ptmprotein/Q9Y463{[pY]@273}{[aR]@277},Q13627{[pY]@321}{[aR]@325},Q5TA31{[meR]@109}

Example with mass difference notation

http://pinet-server.org/pinet/api/phosphoandptm/organism/9606/ptmprotein/Q9Y463{[Y+79.966]@273}{[R+42.037]@277},Q13627{[Y+79.966]@321}{[R+42.037]@325},Q5TA31{[R+14.027]@109}

  For Preotein Modification Query

Search by Mass and Delta, http://pinet-server.org/pinet/api/proteinptmbymass/{mass}/delta/{delta}
http://pinet-server.org/pinet/api/proteinptmbymass/80/delta/0.5
Search by UNIMOD of MOD ids, http://pinet-server.org/pinet/api/proteinptm/{mod}
http://pinet-server.org/pinet/api/proteinptm/UNIMOD:1255
http://pinet-server.org/pinet/api/proteinptm/MOD:00912
Search by specifications, http://pinet-server.org/pinet/api/proteinptmbydescription/{description}
http://pinet-server.org/pinet/api/proteinptmbydescription/phospho
http://pinet-server.org/pinet/api/proteinptmbydescription/serine

  For Kinase Network:

http://pinet-server.org/pinet/api/kinase/genes/{geneList} is sent for the query in the URL, e.g.
http://pinet-server.org/pinet/api/kinase/genes/OCIAD1,NCOR2,ZC3HC1,RPS6KA3,PDPK1,PDPK2P,HN1,BRAF,TP53

  For Gene-Pathway Associasion:

http://pinet-server.org/pinet/api/network/genes/{geneList} is sent for the query in the URL, e.g.
http://pinet-server.org/pinet/api/network/genes/OCIAD1,NCOR2,ZC3HC1,RPS6KA3,PDPK1,PDPK2P,HN1,BRAF,TP53

  For Protein Accession Id Query:

http://pinet-server.org/pinet/api/api/uniprotdb/organism/{organism}/accession/{protein} is sent for the query in the URL, e.g.
http://pinet-server.org/pinet/api/uniprotdb/organism/9606/accession/O15393
