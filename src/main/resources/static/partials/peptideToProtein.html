This document describes the REST APIs provided by piNET. These APIs are for developers who want to programmatically analyze their data using pinet APIs. In this documentation, we will use Python and urls to demo the API requests. The code can be used as is or as a template for your own modifications in Python or another language.

  For Search for a List of Peptides: 


import requests
PINET_URL_BASE = 'http://eh3.uc.edu/pinet/api/'
PINET_URL_LINEAGE = '9606'
PEPTIDE_API_EXTENSION_1 = 'peptide/organism/'
PEPTIDE_API_EXTENSION_2 = '/peptides/'
TIMEOUT = 120
RETRIES = 3


def map_peptides(params):
    url = PINET_URL_BASE + PEPTIDE_API_EXTENSION_1 + PINET_URL_LINEAGE + PEPTIDE_API_EXTENSION_2 + params

    peptide_json = {}
    try:
        peptide_json = _request(url)
    except:
        print "error in map_peptides"
        return peptide_json

    return peptide_json

def _request(url, params=None):
    response = requests.get(url)
    return response.json()

if __name__ == '__main__':
    # Example
    # http://eh3.uc.edu/pinet/api/peptide/organism/9606/peptides/KSAPATGGVKKPHR,IYQYIQSR
    print map_peptides("KSAPATGGVKKPHR,IYQYIQSR")

Protein Line Notation (PLN)

About PLN format

Protein Line Notation (PLN) is a notation that encodes “protein forms” as a linear string of characters. A protein form is a “splice isoform, mutation variant and co- or post-translationally modified form (and combinations thereof) that can arise from translation from [a] PCG,” as defined by the PRO ontology (Natale 2011). It is based on the open-world assumption and can encode both positive and negative information.
