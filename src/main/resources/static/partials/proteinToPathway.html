This document describes the REST APIs provided by piNET. These APIs are for developers who want to programmatically analyze their data using pinet APIs. In this documentation, we will use Python and urls to demo the API requests. The code can be used as is or as a template for your own modifications in Python or another language.

  For Kinase Network:

http://eh3.uc.edu/pinet/api/kinase/genes/{geneList} is sent for the query in the URL, e.g.
http://eh3.uc.edu/pinet/api/kinase/genes/OCIAD1,NCOR2,ZC3HC1,RPS6KA3,PDPK1,PDPK2P,HN1,BRAF,TP53

  For Gene-Pathway Associasion:

http://eh3.uc.edu/pinet/api/network/genes/{geneList} is sent for the query in the URL, e.g.
http://eh3.uc.edu/pinet/api/network/genes/OCIAD1,NCOR2,ZC3HC1,RPS6KA3,PDPK1,PDPK2P,HN1,BRAF,TP53

  For Protein Accession Id Query:

http://eh3.uc.edu/pinet/api/api/uniprotdb/organism/{organism}/accession/{protein} is sent for the query in the URL, e.g.
http://eh3.uc.edu/pinet/api/uniprotdb/organism/9606/accession/Q13627
