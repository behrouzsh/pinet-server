PTMs2Modifiers workflow can be used to map and visualize upstream modifying enzymes that target post-translational modification (PTM) sites in query proteins. These PTM sites are mapped automatically from peptide moieties with PTMs when using the Peptide2Protein workflow.
Enter Uniprot protein IDs with PTM sites specified using PSIMOD, UNIMOD, mass difference or shorthand notations and their (optional) abundance values. Enter Proteins/PTM Proteins or for example try the list of LINCS P100 associated PTM Proteins  PTM/Proteins can be separated by comma, semi-colon or new line.

This document describes the REST APIs provided by piNET. These APIs are for developers who want to programmatically analyze their data using pinet APIs. In this documentation, we will use Python and urls to demo the API requests. The code can be used as is or as a template for your own modifications in Python or another language.

import requests
PINET_URL_BASE = 'http://eh3.uc.edu/pinet/api/'
PINET_URL_LINEAGE = '9606'
PEPTIDE_API_EXTENSION_1 = 'phosphoandptm/organism/'
PEPTIDE_API_EXTENSION_2 = '/ptmprotein/'
TIMEOUT = 120
RETRIES = 3



def map_ptms(params):
    url = PINET_URL_BASE + PEPTIDE_API_EXTENSION_1 + PINET_URL_LINEAGE + PEPTIDE_API_EXTENSION_2 + params

    peptide_json = {}
    try:
        peptide_json = _request(url)
    except:
        print "error in map_peptides"
        return peptide_json

    return peptide_json

def _request(url, params=None):
    response = requests.get(url)
    return response.json()

if __name__ == '__main__':

Nodes representing individual sites are color coded to indicate their abundance, if provided.

PhosphoSitePlus

Phosphorylation site-specific kinase network (PSSKN) shows connections between kinases and their target sites in query peptides/proteins.
Solid lines in the graph represent known relations between kinases and their target sites (in Known_Kinase_TargetGene), whereas dashed lines represent predicted relationships using consensus sequences derived from PhosphoSitePlus. Two different prediction modes can be used, based on Position Weight Matrices for consensus motifs (Known+Predicted_Probability_Kinase_TargetGene), or simply using sequence similarity between known targets and query peptides (Known+Predicted_Blosum50_Kinase_TargetGene), with the latter providing predictions even when very few examples of known phosphorylation sites are available.
